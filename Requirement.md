# サービス要件

## サービス概要

- ガイドブックのデジタル版として制作
- マップなどを用い、建築やルートを案内するサービス

## 実装したい機能

- マップのピンを表示
  - 経緯度情報を任意のファイルから読み込み、マップ上にピンを落とす
- 現在地から近いピン情報(=他の建築)をサジェスト
  - 現在地が移動された場合に、都度更新
  - おすすめする建築の近さの最高値、または現在地から他の建築を探すエリア半径を下辺にできるようにする。
- ピンされている建築の詳細情報の表示
  - 基本情報
    - 建物タイトル
    - 設計者
    - 竣工年
    - 住所
    - （写真）
  - 足跡カウント
    - そこを訪れた人の累積人数
- 訪れた建物のリスト機能
  - GPSで特定の建物の近傍に行くと、アプリ側で行った記録を保存。（アチーブメント）

## 取り扱い情報

- JSON
  - `./data`に配置
  - `init.json`
    - ピンを落とすデータを想定して作成
    - タイトル・設計者・経緯度
  - `main.json`
    - 建築ごとのページなどの表示を想定して作成
    - 詳細情報全て
    - タグ：`tags`内に配列で配置。タグ検索・表示などに使用する想定。
- 画像
  - 準備中
